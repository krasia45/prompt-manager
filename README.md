# 📌 나만의 프롬프트 관리 프로그램 (Prompt Manager)

> 파이썬(Python) 콘솔 기반으로 자주 사용하는 AI 프롬프트를 체계적으로
> 분류, 검색, 관리하고 Git을 통해 변경 이력을 관리하는 프로젝트입니다.

------------------------------------------------------------------------

# 🚀 실행 방법

1.  레포지토리를 클론하거나 프로젝트 폴더로 이동합니다.
2.  터미널에서 아래 명령어를 실행하여 프로그램을 시작합니다.

``` bash
python main.py
```

3.  화면에 출력되는 메뉴 번호를 입력하여 원하는 기능을 사용합니다.
4.  종료하려면 `0`을 입력합니다.

------------------------------------------------------------------------

# 🛠️ 기능 목록

## 1. 프롬프트 추가

새로운 프롬프트의 제목, 내용, 카테고리를 입력하여 등록합니다.

## 2. 프롬프트 목록

저장된 전체 프롬프트 목록을 번호와 함께 출력하며, 즐겨찾기 여부(★)를
표시합니다.

## 3. 카테고리별 조회

카테고리별로 저장된 프롬프트를 모아서 확인합니다.

## 4. 프롬프트 검색

키워드를 입력하여 제목 또는 내용에서 원하는 프롬프트를 검색합니다.

## 5. 프롬프트 상세 보기

번호를 입력하여 프롬프트의 전체 내용을 확인합니다.

## 6. 즐겨찾기 관리

즐겨찾기를 추가하거나 해제합니다.

## 7. 즐겨찾기 목록

즐겨찾기로 등록된 프롬프트만 확인합니다.

## 0. 종료

프로그램을 안전하게 종료합니다.
