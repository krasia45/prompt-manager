# 🚀 나만의 프롬프트 관리 프로그램 (Prompt Manager)

GenAI 기초부터 제작한 다양한 프롬프트(텍스트, 이미지, 페르소나 등)를 한곳에서 체계적으로 분류하고 관리하는 Python 콘솔 기반 어플리케이션입니다.

## 🛠️ 주요 기능
* **프롬프트 추가**: 제목, 내용, 카테고리를 지정하여 새로운 프롬프트 등록 (빈칸 예외 처리 포함)
* **프롬프트 목록**: 등록된 모든 프롬프트의 카테고리, 제목, 즐겨찾기(⭐) 여부 확인
* **카테고리별 조회**: 선택한 특정 카테고리의 프롬프트만 필터링하여 조회
* **키워드 검색**: 제목 또는 내용에 검색어가 포함된 프롬프트 탐색
* **상세 보기**: 특정 프롬프트의 전체 내용과 세부 정보 확인
* **즐겨찾기 관리**: 프롬프트별 즐겨찾기 토글(추가/해제) 및 즐겨찾기 목록 모아보기

## 📂 등록된 프롬프트 카테고리
* 텍스트 생성 / 이미지 생성 / 영상 생성 / 페르소나 / 자동화 / 기타

##확인 및 실행: 엔터를 치면 1초 만에 깃허브에 있던 파일들이 왼쪽에 쫙 나타납니다. 그대로 python main.py를 입력해서 다시 연습해 보시면 됩니다.

### 1. VSCode에서 Python 파일 생성 및 실행
* **VSCode**: 가볍고 확장성이 뛰어난 마이크로소프트의 코드 에디터입니다.
* **생성 및 실행**: `.py` 확장자를 가진 파일을 만들고, 터미널창에서 `python 파일이름.py` 명령어를 입력해 스크립트를 실행할 수 있습니다.

### 2. 터미널에서 Python/Git 버전 확인 및 설정 점검
* **버전 확인**: `python --version` 또는 `python3 --version`, 그리고 `git --version`을 통해 개발 도구가 올바르게 설치되었는지 확인합니다.
* **설정 점검**: `git config` 명령어로 커밋 시 기록될 사용자 이름과 이메일을 세팅합니다.

### 3. 파이썬 기초 문법
* **변수와 자료형**: 데이터를 저장하는 공간(문자열, 정수, 불리언 등).
* **조건문 (`if/else`)**: 상황에 따라 분기 처리를 수행하는 문법.
* **반복문 (`for/while`)**: 동일한 로직을 반복 수행하는 문법.
* **함수 (`def`)**: 재사용 가능한 코드 블록 정의.

### 4. Git이 무엇이고 왜 필요한지
* **Git**: 소스 코드의 변경 이력을 시간순으로 기록/추적하는 분산 버전 관리 시스템입니다.
* **필요성**: 문제 발생 시 안전한 시점으로 롤백이 가능하고, 여러 사용자가 충돌 없이 협업할 수 있도록 돕습니다.

### 5. init, add, commit, push, pull, checkout, clone, merge의 역할
* **`git init`**: 로컬 디렉토리를 Git 저장소로 변환.
* **`git add`**: 변경된 파일을 스테이징 영역(대기열)에 등록.
* **`git commit`**: 스테이징된 파일들의 상태를 스냅샷 형태로 기록.
* **`git push`**: 로컬 커밋 이력을 원격 저장소에 업로드.
* **`git pull`**: 원격 저장소의 최신 변경 내용을 가져와 로컬에 반영.
* **`git checkout`**: 특정 브랜치나 커밋 시점으로 이동(전환).
* **`git clone`**: 원격 저장소 프로젝트를 내 PC로 복제.
* **`git merge`**: 다른 브랜치의 내역을 현재 브랜치로 병합.

### 6. 브랜치 생성 및 병합
* **브랜치(Branch)**: 메인 코드에 영향을 주지 않고 독립적으로 새 기능을 개발하는 공간 (`git branch`, `git checkout -b`).
* **병합(Merge)**: 개발이 완료된 브랜치를 메인 브랜치로 합치는 작업 (`git merge`).

### 7. GitHub에 코드 업로드 및 관리
* **GitHub**: 원격 저장소를 제공하며 오픈소스 공유 및 협업을 지원하는 웹 서비스. `git remote add origin` 및 `git push`를 통해 코드를 업로드하고 관리합니다.
